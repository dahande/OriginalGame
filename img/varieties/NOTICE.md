# Apple variety photos — NOTICE

This directory contains apple photographs sampled from the
**Fruit-Images-Dataset** (Fruits-360) by Mihai Oltean and Horea Muresan,
distributed under the MIT License.

> Copyright (c) 2017-2020 Mihai Oltean and Horea Muresan
> https://github.com/Horea94/Fruit-Images-Dataset

The original dataset uses generic apple variety names (e.g. *Red 1*,
*Pink Lady*, *Granny Smith*). For this game the closest-color variety
from the dataset has been mapped to each Aomori cultivar:

| File              | Aomori variety   | Source folder            |
| ----------------- | ---------------- | ------------------------ |

| senshu.jpg        | 千秋             | Apple Red 1              |
| tsugaru.jpg       | つがる           | Apple Red Yellow 1       |
| kio.jpg           | きおう           | Apple Golden 1           |
| jonagold.jpg      | ジョナゴールド   | Apple Red Yellow 2       |
| toki.jpg          | トキ             | Apple Golden 2           |
| shinano-sweet.jpg | シナノスイート   | Apple Crimson Snow       |
| ourin.jpg         | 王林             | Apple Granny Smith       |
| mutsu.jpg         | 陸奥             | Apple Golden 3           |
| fuji.jpg          | ふじ             | Apple Pink Lady          |
