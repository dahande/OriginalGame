# 青森りんご合体 🍎 (Aomori Apple Merge)

青森の代表 11 品種を 全部使った 落ちもの 物理パズル。
スイカゲーム風に、同じ品種の りんごを ぶつけて 合体させ、最終的に **世界一** を 目指す。

![play](https://img.shields.io/badge/play-browser-c0271e) ![tech](https://img.shields.io/badge/tech-canvas%20%2B%20webaudio-3f8c3f) ![license](https://img.shields.io/badge/license-MIT-efc94c)

## 進化チェーン ( 11 段 )

1. 🍎 **紅玉** ( こうぎょく )
2. 🍎 **千秋** ( せんしゅう )
3. 🍎 **つがる**
4. 🍎 **きおう**
5. 🍎 **ジョナゴールド**
6. 🍎 **トキ**
7. 🍎 **シナノスイート**
8. 🍎 **王林** ( おうりん )
9. 🍎 **陸奥** ( むつ )
10. 🍎 **ふじ**
11. 🏆 **世界一** ( せかいいち )

ドロップ可能なのは **下位 4 品種** ( 紅玉〜きおう )。それより 大きい品種は 合体でしか 生まれません。

---

## 遊び方

1. 上部で マウス / 指を 動かして 投下位置を 決める
2. クリック / タップで りんごを 落とす
3. 同じ品種の りんごが ぶつかると 一段上に **合体**
4. **赤い ライン** より 上に りんごが 1.8 秒 留まると ゲームオーバー
5. **世界一** を 作れたら 大成功

### 操作

- マウス / タッチ : 投下位置の指定 / 投下
- `R` キー : ゲームオーバー時に リスタート
- 右上 🔊 : サウンド切替

## ローカルで 動かす

ES Modules を 使っているので、HTTP サーバー経由で 起動。

```bash
python3 -m http.server 8080
# あるいは
npx serve -l 8080 .
```

ブラウザで `http://localhost:8080/`

## 実装メモ

- **依存ゼロ** : npm パッケージ・物理ライブラリ なし、ブラウザ標準 API のみ
- **円同士の物理** : 自前で 重力・反発・衝突解決 ( 位置補正 + 速度インパルス )
  を 実装。サブステップ × 4 で 安定化。
- **合体** : 同ティア が 接触すると、両者を 削除して 中点に 一段上の りんごを
  生成し、合体時の 平均速度 + 軽い上方インパルスを 与える。
- **ゲームオーバー判定** : 投下から 1.2 秒の グレース後、危険ライン上に
  速度小で 1.8 秒 留まると 終了。
- **音** : Web Audio で その場合成。合体時は 進化段に 応じて ペンタトニックの
  音が 上がっていく。
- **リアルな りんご描画** : 起動時に **Wikimedia Commons API** で 各品種を
  検索し、見つかった 写真を `<img crossOrigin="anonymous">` で 取得して
  円形クリップ + 影 + ハイライトで 球体感を 出す。検索が ヒットしない
  品種は ハードコードされた `Special:FilePath` URL候補へ フォールバック、
  さらに 全部 失敗したら 自前の Canvas 描画 ( 多段グラデ + 縞・斑点・
  カリックス + 軸 + 葉 ) を 表示する 三段構え。
  画像は 各品種ごとに オフスクリーンキャンバスに 一度だけ レンダして
  キャッシュ。
- **危険ライン判定** : 投下後 0.8 秒の グレース後、ラインを 超えて
  1.6 秒で ゲームオーバー。一度 ラインを 越えた 後は ヒステリシス
  ( ライン下 8px ) で 振動による 誤リセットを 防止。
